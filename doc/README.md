# 文档目录索引

本目录按文档用途分组，避免需求、设计、演示和过程规划混在根目录。

## 目录结构

| 目录 | 内容 | 说明 |
|------|------|------|
| `requirements/` | 需求、背景、修改意见、字段资料 | 项目输入材料和业务约束 |
| `design/` | 业务逻辑设计 | 业务流程、处理规则、场景设计 |
| `guides/` | 启动与使用指南 | 本地运行、演示操作、常见问题 |
| `demo/` | Demo 讲解稿 | 汇报讲解、演示口径、产品说明 |
| `planning/` | 任务计划、发现、进度 | 持续规划与执行记录 |
